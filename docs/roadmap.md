# Roadmap

The planned development path for goAuthly.

## Phase 1: Foundation (Current)

- [x] Initial repository structure and documentation.
- [ ] Core authorization verification logic.
- [ ] Basic error handling framework.

## Phase 2: Core Functionality

- [ ] OAuth 2.0 token verification implementation.
- [ ] JWK fetching and caching logic.
- [ ] Basic authentication support.

## Phase 3: Framework Integration

- [ ] Fiber adapter implementation.
- [ ] gRPC interceptor for authorization.
- [ ] FastHTTP middleware.

## Phase 4: Optimization and Polishing

- [ ] Performance benchmarking and optimization.
- [ ] Comprehensive documentation and examples.
- [ ] Stabilization and v1.0.0 release.
